⚠️ 주의점

🎬 스크롤 복원 동작 (3안 공통)

  1. 메인 / 자유게시판 목록에서 사용자가 스크롤하다가 5번째 카드 클릭
  2. onClick 핸들러가 sessionStorage.setItem('home-feed-scroll', window.scrollY)
  3. 게시글 상세 페이지로 이동
  4. "뒤로" 버튼 클릭 → router.back()
  5. 목록 페이지가 다시 마운트되면서 useEffect에서 window.scrollTo(0, sessionStorage.getItem('home-feed-scroll'))
  6. 5번째 카드 위치로 정확히 복원 ✨

⚠️ 새 탭/새 윈도우는 sessionStorage 분리되어 복원 안 됨 — 의도된 동작 (새 탭은 새 컨텍스트).

📌 최종 추천: A안

콘텐츠 영역 좌상단에 텍스트 버튼 (모바일/데스크톱 동일).

구현 단순도가 높고 (BackButton 컴포넌트 1개 + sessionStorage 훅 1개), 다른 페이지 영향 없음, 학습 비용 0이며, 게시글 같은 콘텐츠 페이지에 가장 어울리는 패턴입니다. 적용 위치: /posts/[id], /board/[id].
