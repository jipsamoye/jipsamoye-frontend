E 그리미티 스타일 + 실제 펫 사진

카드 위에 큰 popover. 가운데 실제 프로필 사진(amber ring) + bio + 받은하트/구독자/랭킹 + 구독하기/메시지. 시각 인상 가장 강함.

✨ E 옵션의 특징

  • • 카드 위로 떠오르는 popover: 원본 카드 위치에 그대로 등장 → 시각 흐름 자연스러움
  • • 가운데 실제 펫 사진: amber ring으로 강조 (그라데이션 배경 X) → 사용자가 올린 진짜 사진을 부각
  • • 펫 프로필 페이지 미니어처: 받은하트/구독자/랭킹 + 구독하기/메시지 → 클릭 안 해도 핵심 정보·행동 가능
  • • 구현 비용: 중간~큼 (~200줄). popover 위치 조정 + 외부 클릭/ESC 닫기 디테일 필요
  • • 주의: popover가 카드를 가림 → 카드 자체를 클릭하려는 의도와 충돌 가능. 400ms 지연 + popover 영역 hover 유지 등 디테일로 완화

💡 제 추천: C · Hovercard (펫 프로필 요소)

  • • 펫 프로필 페이지의 핵심 요소(받은하트·구독자·랭킹 + 구독하기·메시지)를 그대로 압축 → 사이트 디자인 일관성
  • • 카드 안에서 *알아보고 → 행동* 까지 한 번에 (프로필 페이지로 안 가도 구독·DM 가능)
  • • 한 번 만들면 PostCard·PopularSlider·댓글·DM 목록 등 *닉네임이 보이는 모든 곳*에 재사용
  • • 단, totalLikeCount / ranking 백엔드 추가 필요. 백엔드 작업 전이면 임시로 "-" 표시 또는 D 옵션 우선 적용 가능
  • • 디테일: hover 250ms 지연, popover 영역 hover 시 유지, viewport 끝 가까우면 위로 반전, 모바일은 1탭=hover/2탭=실행 또는 long-press
