인스타급. 사람 부각 강함. 단 사진 카드 정보 위계가 살짝 흔들 — 작가가 사진과 동급으로 인지될 수 있음.

💡 제 추천: 24px · w-6 h-6

  • • 닉네임 폰트(12-14px)보다 1.5~2배 — 프로필이 *주체*로 인지되는 골디락스 사이즈
  • • 사진 컨텐츠 영역 손실 0 — 메타 영역 안에서만 자라남
  • • 토스/그리미티/핀터레스트 카드 하단 프로필 표준 — 학습 비용 0
  • • 카드 height 변화 거의 없음 — 메타 영역 line-height가 16~20px라 24px 프로필이 들어가도 추가 padding 미미
  • • 28px도 좋은 선택지 — 사람 강조 톤 원하면 한 단계 더. 32px는 인스타급이라 사진 자랑 사이트에는 살짝 강함
